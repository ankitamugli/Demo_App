# Demo-App

A simple Java JDBC project that demonstrates how to connect Java with MySQL database using JDBC.

## 🚀 Technologies Used

- Java
- JDBC
- MySQL
- Eclipse IDE
- Git & GitHub

---

## 📌 Features

- Load MySQL JDBC Driver
- Connect Java Application with MySQL Database
- Create SQL Statements
- Execute Insert Query
- Exception Handling using Try-Catch
- Basic JDBC CRUD Operations
